# Stół do TTRPG

## Autor
Oliwier Pietrasik

## Opis projektu
Stół do TTRPG to prosta aplikacja internetowa umożliwiająca dodawanie żetonów na planszę, przesuwanie ich po siatce oraz wykonywanie rzutów kośćmi.

## Funkcjonalności
- Dodawanie nowych żetonów na planszę
- Przesuwanie żetonów metodą „przeciągnij i upuść”
- Automatyczne przyciąganie żetonów do siatki 50x50 px
- Rzuty kośćmi:
  - K4
  - K6
  - K8
  - K12
  - K20
- Wyświetlanie wyniku ostatniego rzutu

## Technologie
- HTML5
- CSS3
- JavaScript

## Struktura projektu
- `index.html` – struktura strony
- `style.css` – wygląd aplikacji
- `script.js` – logika programu
